☁️ Sync across devices (via GitHub)

Not connected — data is saved on this device only.
First device (one-time):
1. Create a GitHub token at github.com/settings/tokens → Tokens (classic) → Generate new token → tick only the gist scope → copy it.
2. Paste it above and hit Save, then click ➕ Create sync file. A private Gist is created and its ID appears above.

Every other device (phone, laptop):
Open this same page, go to Sync, paste the same token and the same Gist ID, hit Save on both, then ⬇️ Pull from GitHub. After that, edits auto-save to GitHub and every device stays in sync (Pull when you open it on another device).

🔒 Your token is stored only in this device's browser — never in the app code or the public site. Use a token limited to the gist scope. The data Gist is private.
